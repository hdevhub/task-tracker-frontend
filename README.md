# Task Tracker Frontend

## Mục tiêu
Dự án web đơn giản giúp người dùng quản lý công việc hằng ngày.
Mục đích chính là luyện HTML, CSS và JavaScript cơ bản theo hướng thực tế.

## Công nghệ sử dụng
- HTML5
- CSS3
- JavaScript (Vanilla)

## Tính năng dự kiến
- Thêm / xoá / sửa task
- Đánh dấu task đã hoàn thành
- Lưu dữ liệu bằng localStorage
- Giao diện đơn giản, dễ dùng
